## Проблемы и решения в процессе разработки

### 1. SSH — не удавалось подключиться к серверу
- **Проблема:** Пароль не принимался при подключении по SSH
- **Решение:** Создан SSH-ключ (ed25519), добавлен на сервер через консоль TimeWeb
- **Файл ключа:** `C:\Users\Пользователь\.ssh\ilmavent`

### 2. Дашборд не открывался (404)
- **Проблема:** Папка `public/` не была загружена на сервер
- **Решение:** Загружена через SCP, перезапущено приложение

### 3. Порт 3000 не доступен извне
- **Проблема:** Фаервол блокировал порт
- **Решение:** Открыт порт через `ufw allow 3000` и `iptables`

### 4. Квизы «не работают»
- **Проблема:** При тесте квизов новая заявка не появлялась
- **Причина:** Использовался тот же номер телефона → система определяла дубль
- **Решение:** Не баг, а фича — защита от дублей работает корректно
